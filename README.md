# 星尘沙盘
## 自由探索沙盘类游戏

星尘沙盘是一个大型联机沙盒游戏

## 核心玩法

玩家可以自行创造任何物品、产业、文明、政府、企业，体验新的人生。也可以加入其他玩家创建的产业、文明、政府、企业的同时游戏具有剧情、感情线、事业线等内容。
