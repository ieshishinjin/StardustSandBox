# ssb项目结构分析

## ssb目录结构

```
ssb/
├── .vscode/               # VS Code配置文件
│   ├── extensions.json    # 扩展推荐配置
│   ├── launch.json        # 调试配置
│   └── settings.json      # 编辑器设置
├── Assets/                # 游戏资源主目录
│   └── Scenes/            # 游戏场景文件
│   └── angle.cs           # 移动视角脚本
│   └── player.cs          # 移动脚本
├── Logs/                  # 日志文件
├── Packages/              # Unity包管理
│   ├── manifest.json      # 包清单
│   └── packages-lock.json # 包锁定文件
├── UserSettings/          # Unity用户设置
│   ├── Layouts/           # 编辑器布局
│   │   └── CurrentMaximizeLayout.dwlt # 当前最大化布局
│   │   └── default-2022.dwlt         # 默认2022布局
│   └── Search.index       # 搜索索引文件
│   └── Search.settings    # 搜索设置
├── .gitignore             # Git忽略文件
├── 项目结构.md            # 本项目结构文档
├── Assembly-CSharp.csproj    # C#项目文件
└── ssb.sln           # Visual Studio解决方案文件
```
